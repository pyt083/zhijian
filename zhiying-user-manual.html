1产品概述

什么是智剪？

智剪是一款面向内容创作者和运营团队的AI 批量视频混剪平台。它集成了智能切片、智能打标、智能脚本生成、AI 生图、AI 生视频、字幕擦除、画质增强等核心能力，帮助用户从原始素材到成片输出实现全流程自动化。


产品名称智剪（Insai InVision）
产品定位批量视频智能混剪工具 + AI 创作工作台
访问方式企微工作台 → 业务运营赋能 → 智剪
计费体系星火币（余额自动同步）
支持格式MP4 / MOV / PNG / JPEG 等

核心能力一览

🔪 智能切片

自动识别视频中的精彩片段（高光对话、动作场景、卖点时刻），一键拆分。

🏷️ 智能打标

为素材自动添加场景、情绪、物体标签，便于快速检索与分类。

📜 脚本生成

根据素材内容自动生成剪辑脚本，含转场、BGM、字幕建议。

🎬 批量混剪

一键合并数百条片段，支持预设模板（带货/Vlog/活动回顾等）。

🖼️ AI 生图

文字描述或参考图一键生成营销图片，多比例、多格式、多质量可选。

🎥 AI 生视频

输入文案/素材，AI 自动生成短视频，支持多种分辨率和时长。

8视频处理工具

智剪提供一系列视频预处理工具，帮助你在进入混剪流程前优化原始素材。

8.1 单视频抽帧裂变

功能说明：对单个视频进行逐帧抽帧并裂变为多个输出变体，适用于从一条长视频中批量提取关键帧画面。

操作步骤

  1. 进入「智剪素材裂变平台」→ 选择「单视频抽帧裂变」标签页
  2. 点击「选择主视频」上传源文件（MP4/MOV等）
  3. 设置输出目录路径
  4. 填写任务名称、裂变数量、每秒抽取帧数
  5. 选择业务标签和素材类型（如：剪辑）
  6. 点击「开始抽帧裂变」，右侧面板实时显示运行日志

图 8-1：单视频抽帧裂变操作界面

⚠️ 注意：抽帧过程依赖 FFmpeg 运行时环境，若出现「Error initializing filters」错误，请检查视频编码参数是否兼容。

8.2 智能切片

功能说明：系统自动识别视频中的场景切换点和高光时刻，将长视频切分为若干有意义的片段。

操作入口

顶部导航栏 → 「切片」 Tab 页

图 8-2：智能切片工作台 —— 上传视频后自动识别片段

  1. 在左侧「所属项目」下拉框中选择目标项目
  2. 拖拽或点击上传视频文件（支持 MP4/MOV）
  3. 系统自动执行切片分析，左侧列表展示已生成的切片任务
  4. 右侧预览区可播放查看每个切片片段
  5. 点击右上角切片列表可进行全选/批量导出
💡 提示：切片灵敏度可在工具设置中调整——值越高，切割出的片段越多越细。

8.3 画质增强

功能说明：利用 AI 超分辨率技术提升视频/图片的清晰度，支持 4K 输出。

图 8-3：画质增强模块 —— 处理前 vs 处理后对比

使用方法

  1. 顶部导航进入「高清增强」页签
  2. 选择增强模式：专业版 或 AIGC 视频
  3. 设置目标分辨率（如 4K）及宽高像素值
  4. 设置日帧率参数
  5. 上传视频，系统自动处理并在右侧显示前后对比效果

8.4 字幕擦除

功能说明：AI 自动检测并去除视频中的硬字幕/水印区域，还原干净画面。

  1. 导航至「字幕擦除」功能页
  2. 选择擦除模式： 字幕擦除 字幕擦除 文字擦除

9快速上手流程

🚀 从零产出第一条视频（5 步走）

  1. 创建项目 → 进入「项目」Tab → 「+ 新建项目」→ 填写项目名和基本信息
  2. 上传物料 → 项目内切换到「物料」Tab → 上传原始视频和参考图片
  3. 智能切片 → 进入「切片」Tab → 选择项目 → 上传视频 → 自动生成切片片段
  4. 创建任务 → 进入「任务」Tab → 「+ 一键创建」→ 选项目/填要求/设时长分辨率
  5. 提交生成 → 查看 AI 方案 → 确认后提交 → 右侧「混剪作业」等待完成 → 下载成品
💡 效率技巧：善用「脚本」功能先让 AI 出方案，确认创意方向后再跑任务，可大幅减少无效生成消耗的星火币。

10常见问题

Q1：星火币余额不足怎么办？

星火币由公司统一发放，通过企微工作台自动同步。若余额不足，请联系部门管理员申请额度。

Q2：任务生成失败怎么排查？

在生成状态面板会显示具体错误信息（如分辨率不支持、内容违规等）。根据提示调整参数后重新提交即可。

Q3：如何提高成片质量？

① 使用更清晰的原始素材；② 在创意要求中详细描述画面风格和节奏；③ 选择更高分辨率和质量档位；④ 上传参考图锁定风格一致性。

Q4：支持哪些视频格式导入？

主流格式均支持：MP4、MOV、AVI、MKV 等。建议优先使用 MP4 (H.264/H.265 编码) 以获得最佳兼容性。

Q5：成品保存在哪里？

自动保存至项目的「成品」Tab 下，也同步到个人资产库。支持在线预览和批量下载。
